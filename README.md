# Question Answering and Chatbot Systems

---

## Project Overview
This repository contains implementations for **Lab 9**, covering multiple NLP tasks using `transformers`, `deep_translator`, and `openai` libraries.  
It demonstrates:
- Building Question Answering (QA) systems.
- Handling unanswerable questions.
- Creating English-to-Arabic translation chatbots (rule-based and LLM-based).
